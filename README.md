# CMO-prototype-framework-text-edition
This is a prototype framework for the edition of ottoman lyric poetry.

The framework is oXygen-based. Add it as an external framework to your oXygen editor.
It will apply to all files ending with *_cmo.xml, *.cmo.xml, *_CMO.xml and *.CMO.xml

It contains a reduced scheme and customizations of the GUI.
